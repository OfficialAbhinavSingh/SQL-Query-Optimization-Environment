## Before/after (environment-only contrast)

To reproduce the “showing improvement” table without any API keys:

```bash
python training/eval_before_after.py --save-dir results
```

This does **not** retrain a model; it contrasts a **deliberately weak** action (no optimization) against the **hand-crafted** fallback on identical tasks so judges can see the reward spread attributable to real DuckDB execution.
